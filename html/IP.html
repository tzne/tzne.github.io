1. Introdução

  • Ciência da computação: ciência que estuda as técnicas, metodologias e instrumentos computacionais, que automatiza processos e desenvolve soluções baseadas no uso do processamento digital

2. Algoritmos

  • Sequência finita de instruções bem definidas e não ambíguas para resolver um problema ou realizar algum objetivo
  • Programa: sequência de instruções em alguma linguagem compreendida pelo computador

3. Variáveis

  • Locais na memória onde se armazenam informações
    • Apenas um dado por vez
  • Em C, as variáveis devem ser declaradas antes de serem utilizadas
  • Existem apenas em tempo de execução
    • São perdidas ao fim do programa
  • Toda variáveel tem um nome e um tipo
    • int: %d
      • short unsigned int: %u
      • short signed int: %d
      • signed int: %d
      • unsigned int: %u
      • long signed int: %ld
      • long unsigned int: %lu
    • float: %f
      • double: %lf
      • long double: %LF
    • char: %c
      • signed char: %c
      • unsigned char: %c
  • Por segurança, é aconselhado a inicializar as variáveis antes de utilizá-las

3.1. Nomenclatura

  • Deve começar com uma letra ou underscore (_)
  • Não pode começar com um número
  • Pode conter letras maiúsculas, minúsculas, números e underscore
  • Por convenção, não são escritos inteiros em LETRAS MAIÚSCULAS
    • Padrão reservado para constantes
  • Caracteres proibidos
    • {
    • }
    • +
    • =
    • *
    • /
    • \
    • ;
    • .
    • ,
    • ?
    • Acentos
  • Identificadores são case-sensitive
  • Não pode ser uma palavra reservada da linguagem C
  • Devem ser claros

3.2. Conversão de tipos

  • Conversão implícita: capacidade do destino deve ser maior que a origem 

    int a;
short b;
a = b;

  • Conversão expícita: não modifica o tipo da variável, apenas da expressão 

    a = (int) ( (float) b / (float) c);

4. Vetores

  • Coleção de variáveis de mesmo tipo
  • Acesso pelo índice
  • Tamanho pré-definido
    • Após a compilação, não pode ser alterado

  • Declaração: 

    tipo identificador[tamanho];

char cadeia[50];
float pares[100];
float notas[5] = {10.0, 4.1, 7.3, 8.2, 2.7};
int posicoes[3] = {2, 4}; // completa os outros espaços com 0
int posicoes[] = {2, 4};  // cria um vetor de tamanho 2
  • Primeira posição é 0

4.1. Exercício

  • Escreva um programa para receber as notas de p1, p2 e L de n alunos ( 0 < n <= 10)
  • Calcule e exiba
    • a média simples de p1, p2 e L
    • a média de cada aluno, sendo que M = (2p1 + 3p2 + 5L)/10
    • a quantidade de alunos aprovados, reprovados e que ficaram de SAC
#include <stdio.h>

#define MAX 10

int main() {
  float p1[MAX], p2[MAX], l[MAX], media_p1=0, media_p2=0, media_lista=0,
    media_aluno[MAX];
  unsigned int n, aprovados=0, reprovados=0, sac=0;

  scanf("%d", &n);
  for (int i = 0; i < n; i++) {
    printf("Insira as notas do %d° aluno: ", i+1);
    scanf("%f %f %f", &p1[i], &p2[i], &l[i]);
   
    media_p1 += p1[i];
    media_p2 += p2[i];
    media_lista += l[i];
  }

  media_p1 /= n;
  media_p2 /= n;
  media_lista /= n;

  printf("Média P1: %4.1f\n", media_p1);
  printf("Média P2: %4.1f\n", media_p2);
  printf("Média lista: %4.1f\n", media_lista);

  for (int i = 0; i < n; i++) {
    media_aluno[i] = (2*p1[i] + 3*p2[i] + 5*l[i]) / 10;
   
    printf("Média aluno %d: %4.1f\n", i+1, media_aluno[i]);
   
    if (media_aluno[i] < 5) {
      reprovados++;
    } else if (media_aluno[i] < 6) {
      sac++;
    } else {
      aprovados++;
    }
  }

  printf("Aprovados: %d\n", aprovados);
  printf("SAC: %d\n", sac);
  printf("Reprovados: %d\n", reprovados);

  return 0;
}

5. Matrizes

  • Vetor com duas ou mais dimensões

  • Declaração: tipo nome[linhas][colunas] 

    float salas[20][30];

    • Mais de duas dimensões: 

      int dimensoes[4][3][2]; // matriz com 3 dimensões
char pessoas[3][9][7][9]; // matriz com 4 dimensões
float numeros[2][2] = {{1, 2},
                       {3, 4}};
float pares[2][3] = {0, 2, 4, 6, 8, 10};

  • Manipulação 

    salas[19][12] = 4;
    • O compilador não verifica se os valores inseridos são válidos na matriz

6. Redefinição de tipos, registros e tipos enumerados

  • O comando sizeof indica o tamanho em bits de uma variável 

    printf("%u", sizeof(double)) // retorna 8

6.1. Redefinição de tipos

  • Útil quando a alteração do tipo de uma variável altera o tipo de várias variáveis

  • Feito através do comando typdef 

    // normalmente feita fora da main
typedef tipo_primitivo tipo_novo;
typedef float nota; // cria um tipo nota, com variáveis de ponto flutuante

int main() {
  nota p1, trabalho;

  scanf("%f", &p1);
  }

6.2. Registros

  • Variável que contém diversas variáveis (campos)
    • Podem ser de tipos diferentes
    • Faz sentido serem agrupadas

  • Declarados a partir do comando struct 

    // normalmente feita fora da main
struct nome_do_tipo_do_registro {
  tipo nome_campo1;
  tipo nome_campo2;
  tipo nome_campo3;
};

// Exemplo
struct cadastro {
  int idade;
  char sexo;
  float imc;
};

int main() {
  struct cadastro cad;
}

  • Campos são acessados por meio de um "." 

    nome_registro.nome_campo;

struct cadastro cad;
printf("Insira a idade do atleta: ");
scanf("%f", &cad.idade);

// escreve a idade do atleta na tela
printf("%f", cad.idade);

  • A inicialização de registros possui duas formas: 

    struct aluno aluno1 = {831596, 9.9, 5.4, 8.3},
             aluno2 = {.ra = 831596,
                       .nota_lista = 9.9,
                       .nota_p1 = 5.4,
                       .nota_p2 = 8.3};

  • Registros podem ser copiados, assim como variáveis 

    aluno1 = aluno3;

  • É possível redefinir um registro para nomea-lo de acordo com a semântica do que ele representa 

    struct aluno {
  int ra;
  nota nota_p1, nota_p2, nota_trabalho;
  float media;
};

typedef struct aluno Aluno;

// ou

typedef struct aluno {
  int ra;
  nota nota_p1, nota_p2, nota_trabalho;
  float media;
} Aluno;

  • É possível declarar um registro como campo de outro registro 

    typedef float nota;

typedef struct prova {
  nota p1;
  nota p2;
  nota trabalho;
  float media;
} Provas;

typedef struct aluno {
  int ra;
  provas notas;
  nota media;
  } Aluno

  • É possível criar vetores ou matrizes de structs 

    struct nota listas[11];

6.3. Tipos enumerados

  • Tipo que funciona como int, porém está associada a constantes enumeradas

  • comando enum 

    enum nome {constante0, constante1, constante_N};

enum mes { Janeiro, Fevereiro, Marco, Abril, Maio, Junho, Julho, Agosto,
            Setembro, Outubro, Novembro, Dezembro }; // janeiro equivale a 0

  • Declaração de variável 

    enum mes aniversario; // mes é o tipo utilizado

  • Atribuição

    • É possível atribuir um valor para qualquer um dos tipos
    enum mes { Janeiro=1, Fevereiro, Marco, Abril, Maio, Junho, Julho, Agosto,
            Setembro, Outubro, Novembro, Dezembro }; // Janeiro equivale a 1

7. Funções e procedimentos

7.1. Funções e procedimentos

  • Funções são úteis para
    • Evitar blocos muito lognos
    • Separar o programa em partes mais fáceis de se compreender logicamente
    • Reaproveitamento do código
      • Evitar que um trecho seja repetido várias vezes
  • Procedimentos: estruturas que agrupam um conjunto de comandos
  • Funções: procedimentos que retornam um único valor ao fim de sua execução

    • Declaração 

      tipo nome_funcao (tipo parametro1, tipo parametroN) {
  comandos;
  return (valor de retorno);
}
      • Normalmente declaradas após a função main
        • Cria-se um protótipo da função antes
    • Toda função possui um tipo
      • Tipo do valor de retorno
      • Parametros são variáveis iniciadas quando a função é chamada
        • Uma função pode não ter parametros
    • Funções só podem ser declaradas fora de outras funções

    • Exemplo 

      int soma(int a, int b) {
  return (a + b);
  }
    • Cada parâmetro da função possui um parâmetro real
      • Seus valores são copiados para os parâmetros da função
      • O valor dos parâmetros reais não é afetado pelas funções
    • Funções recursivas
  • O tipo void
    • Tipo utilizado especialmente em funções
    • Armazena e retorna conteúdo "indeterminado"
      • Função não retorna nenhum valor
    • Procedimentos na linguagem C são funções do tipo void

7.2. Escopo

  • Variável local: declarada dentro de uma função
    • Existe apenas dentro da função em que foi declarada
    • Ao fim da função, deixa de existir
    • Possui precedencia à variáveis globais
  • Variável global: declarado fora de qualquer função
    • Acessível em todas as funções
    • Existe durante toda a execução do programa

7.3. Registros e arranjos

7.3.1. Registros

  • Registros podem ser passados como parâmetros de uma função
    • Devem ser declarados antes da função
    • O parâmetro formal recebe uma cópia do registro
  • Uma função pode retornar um registro

7.3.2. Arranjos

  • Um arranjo pode ser passado como parâmetro de uma função

    • Geralmente o tamanho do arranjo é tambem passado como parâmetro 

      int maior_elemento(int vetor[], int tamanho) {}
  • Um arranjo não pode ser usado como retorno de uma função
  • Quando utilizados como parâmetros ou valores de retorno, arranjos possuem um comportamento diferente de registros e outros tipos
    • Eles são alterados pela função
      • Ocorre pois quando se é dado um arranjo como parâmetro, são passados por referência
        • O endereço do primeiro endereço é fornecido

7.3.3. Matrizes

  • Quando o arranjo é multidimensional, é necessário informar ao menos o tamanho da segunda dimensão 

    int maior_elemento(int matriz[][2], int linhas, int colunas) {}

8. Strings

  • Toda string é armazenada na memória como vetor de caracteres
  • O fim da string é marcado pelo caractere nulo '\0'

    • Espaços não utilizados também são preenchidos com \0 

      char data[10] = "Julho 06"; // armazenado como "Julho 06\0\0"
  • Strings são referidas pelo endereço de memória do primeiro caractere
  • Em C, caracteres são tratados como inteiros
    • Se referem à tabela ASCII

  • A máscara para strings é %s 

    char str[] = "isso é uma string";

printf("%s", str);

  • O comando puts imprime a string com quebra de linha 

    puts(str);
  • Leitura de strings
    • scanf
      • Insere '\0' ao fim

      • Lê a string até o primeiro caractere em branco 

        scanf("%s", str);

      • Lê até a quebra de linha 

        scanf("%[^\n]", str); // lê até encontrar '\n', subistituindo por '\0'

scanf("%10[^\n]", str); // determina que o número máximo de caracteres é 10

    • gets 

      gets(str);

8.1. Biblioteca string.h

  • Adiciona funções para manipular strings
    • strlen: tamanho da string
    • strcpy: copia a string
    • strncpy: copia n caracteres da string
    • strcat: concatena a string
    • strncat: concatena n caracteres da string
    • strcmp: compara strings
    • strncmp: compara n caracteres da string

8.2. Biblioteca ctype.h

  • Adiciona funções para manipular caracteres
    • tolower: converte caractere para minúsculo
    • toupper: converte caractere para maíusculo
    • islower: checa se caractere é minúsculo
    • isupper: checa se caractere é maiúsculo
    • isalnum: checa se caractere é alfanumérico
    • isalpha: checa se caractere é alfabético
    • isdigit: checa se caractere é numérico
    • isprint, ispunct, isspace: checa se caractere é imprimível, pontuação ou espaço em branco
